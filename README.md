# 🏏 CricMind AI

### Your Personal AI Cricket Intelligence Agent

CricMind AI is a futuristic AI-powered cricket analytics and fantasy assistant platform built using Google Gemini AI.

The platform helps users:
- Generate fantasy cricket teams
- Predict IPL match winners
- Analyze player performance
- Get AI-powered cricket insights
- Interact with an intelligent cricket assistant

Built for the Agentic Premier League (APL) Hackathon.

---

# 🚀 Features

## 🧠 AI Fantasy Team Generator
Generate optimized fantasy cricket teams using:
- Player form
- Pitch analysis
- Match conditions
- Risk strategies

Includes:
- Captain suggestions
- Vice-captain suggestions
- Differential picks

---

## 📊 Match Prediction Engine
Predict cricket match outcomes using AI reasoning.

The AI analyzes:
- Team combinations
- Pitch conditions
- Bowling strength
- Batting depth
- Momentum factors

---

## 🏏 Player Analysis
Analyze player performance and matchups.

Example:
- Virat Kohli vs spin
- Death over specialists
- Powerplay analysis

---

## 🤖 AI Cricket Assistant
Interactive AI assistant for:
- Cricket Q&A
- Match strategies
- Fantasy recommendations
- Tactical insights
